# 🚁 Helicopter Delivery System

## Description
Helicopter Delivery System — это консольное приложение для управления грузовыми вертолетами, экипажами и рейсами.  
Система позволяет хранить и обрабатывать данные о перелетах, контролировать загрузку вертолетов и анализировать эффективность работы экипажей.  

Приложение реализует операции добавления, удаления, редактирования данных, а также выполнение аналитических запросов.

## Installation

### 🔧 Требования
- Компилятор C/C++ (gcc / clang / MSVC)
- SQLite
- Git

### Usage

После запуска программы доступны следующие возможности:

#### Аутентификация
Вход в систему по логину и паролю
Разграничение ролей (администратор / пилот)
#### Управление данными
Добавление, редактирование и удаление:
вертолетов
экипажа
рейсов
#### Аналитика
Статистика по рейсам
Расчет доходов экипажей
Определение наиболее эффективных вертолетов
#### Ограничения
Контроль ресурса вертолета
Проверка корректности вводимых данных
